# Android Application For Medicine Information Management

This Android application is part of an information management system designed for Taipei City Hospital's Traditional Chinese Medicine Department. 
It provides 3 main functions :
- The storage of medicines information using a bar code scanning functionnality
- The management of resupply requests (sending, viewing and editing)
- The execution of picking processes assisted by LCD displays and LED lights using 8 Raspberry Pis

The Android application uses a local MySQL database for storing data and sockets to communicate with the Raspberry Pis.
